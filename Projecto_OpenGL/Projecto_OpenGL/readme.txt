Semestre 2021-1

Este proyecto contiene todos los archivos necesarios para  el funcionamiento del proyecto. Esta
carpeta esta acompañada de algunassubcarpetas, cuyo contenido se describe a continuación:

+ glm/

Incluye la biblioteca correspondiente a glm.

+ include/

Incluye los archivos correspondientes a glew y glfw, así como la biblioteca assimp.

+ lib/
+ Models/
+ shaders/
+ Textures/
